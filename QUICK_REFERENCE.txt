🔧 KEY CALIBRATION STEPS
══════════════════════════════════════════════════════════════════════

ACS712 Current Sensor:
  1. Check module type: 5A / 20A / 30A (check label)
  2. Set sensitivity in code:
     - 5A:  sensitivity = 0.185
     - 20A: sensitivity = 0.100
     - 30A: sensitivity = 0.066

  3. Calibrate zero voltage:
     - Disconnect motor
     - Read analog value with no current flowing
     - Update: zeroVoltage = value

DHT11 Temperature Sensors:
  - Pin 8: Nozzle Temperature
  - Pin 9: Bed Temperature
  - Check wiring for pull-up resistors

⚙️ MODIFYING THRESHOLDS
══════════════════════════════════════════════════════════════════════

Open src/test_with_ml.cpp and find these lines:

const float HEALTHY_NOZZLE_MAX = 225.0;  // ← Edit these values
const float CREEP_NOZZLE_MIN = 235.0;
const float HEALTHY_BED_MAX = 70.0;
const float CREEP_BED_MIN = 72.0;
const float HEALTHY_CURRENT_MAX = 1.8;
const float CREEP_CURRENT_MIN = 2.5;
const float HEALTHY_VIB_MAX = 70;
const float CREEP_VIB_MIN = 90;

How to adjust:
  • Too many false positives ("CREEP" when healthy)?
    → INCREASE creep thresholds (e.g., 2.5 → 2.8)
  
  • Missing creep when it happens?
    → DECREASE creep thresholds (e.g., 2.5 → 2.2)

🐛 TROUBLESHOOTING
══════════════════════════════════════════════════════════════════════

Problem               | Solution
────────────────────────────────────────────────────────────────────
Sensor Error! (OLED) | Check DHT11 wiring, verify 5V power
Current always 0A    | Check ACS712 power and wiring
OLED blank/no data   | Verify I2C address (0x3C), check SDA/SCL
Too many false alarms| Increase threshold values in code
Never detects creep  | Decrease threshold values or recalibrate
Upload fails         | Check board is set to "micro" in platformio.ini

══════════════════════════════════════════════════════════════════════

🔌 HARDWARE PIN MAPPING
══════════════════════════════════════════════════════════════════════

Arduino Micro Pins:
  D8   ← DHT11 (Nozzle Sensor)
  D9   ← DHT11 (Bed Sensor)
  D7   ← Vibration Sensor
  A0   ← ACS712 Current Sensor
  SDA  ← OLED Display (I2C)
  SCL  ← OLED Display (I2C)
  GND  ← Common Ground (all sensors)
  +5V  ← Power for sensors
